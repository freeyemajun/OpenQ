# 初次使用配置说明

操作场景

本文档将对 Discuz! Q 的常用功能进行简要说明，帮助您更好的了解 Discuz! Q 功能项与配置方式，更快的上手 Discuz! Q。

前提条件

  • 已完成 Discuz! Q 部署。
  • 已登录 Discuz! Q 后台。

# 站点信息配置

每一个站点都有自己的独特性，通常情况下会需要根据自己的网站定位、网站品牌、运营模式等去做相应的设置。例如配置显示的站点名称、站点LOGO、增加统计代码便于掌握目前运营情况等。那么您可以在网站的【全局】>【站点设置】下，配置相关信息。如下图所示：

  • 站点名称：会显示在浏览器标题栏或者站点信息页等地方，站点名称是直观描述站点内容和特性的标识，为站点起一个可读性高的名字有助用户的查看和搜索。
  • 站点介绍：对该站点的简单描述。简洁直观的站点介绍有助用户对站点的快速了解。
  • 站点keywords：keywords 特指站点的关键词、关键字。网站的关键词至关重要，关键词代表了网站的市场定位。通常用于站点管理者给站点设定的词汇，以便用户通过搜索引擎能搜到本站点的词汇。
  • 上传图片：

    • 站点logo：设置站点logo可以帮助您通过图像的方式，让用户易于识别和推广的作用，通过形象的图像可以让用户记住站点。

    • 首页头部logo：作用类似于站点logo。展示在手机端首页。

    • 首页头部背景：用于设置手机端首页头部背景图。

    • 站点icon：用于浏览器标题处ico以及分享默认图片。

      更多站点信息配置说明与操作可查看：站点设置 (opens new window)

# 网站备案设置

根据国务院令第 292 号《互联网信息服务管理办法》和工信部令第 33 号《非经营性互联网信息服务备案管理办法》规定，国家对经营性互联网信息服务实行许可制度，对非经营性互联网信息服务实行备案制度。未取得许可或者未履行备案手续的，不得从事互联网信息服务，否则就属于违法行为。

因此，若您的 Discuz! Q 使用中国大陆境内的服务器开办网站必须先办理网站备案，备案成功并获取通信管理局下发的 ICP 备案号后才能开通访问。备案操作指引详情可参考：首次备案 (opens new window)。

备案成功后，您将获取到 ICP 备案编号。您需要将 ICP 备案编号填入【全局】>【站点设置】>【网站备案信息】处。Discuz! Q 将会使您 ICP 备案编号悬挂置网站底部,符合备案开站要求。

# 公安备案设置

您的网站备案申请通过管局审核并获得备案号后，您必须在网站开通后30日内进行公安备案。如果您的网站涉及经营性备案还需申请经营性网站备案许可证。详情可查看：公安备案流程 (opens new window)。

备案成功后，您将获取到公安备案编号。您需要将 ICP 备案编号填入【全局】>【站点设置】>【公安备案信息】处。Discuz! Q 将会使您 公安备案编号悬挂置网站底部,符合备案开站要求。

# 附件设置

Discuz! Q 在支持发表文本内容外，还支持图片、文件的上传。附件设置可以配置允许上传的附件扩展名及最大尺寸。如下图所示：

  • 支持的图片扩展名：特指图片文件的后缀名。 常见的图片格式有BMP（位图）、JPG、JPEG、PNG、GIF、HEIC等。
  • 支持的文件扩展名：特指文件的后缀名.常见的文件格式有DOC、DOCX、PDF、ZIP等。
  • 支持的最大尺寸：特指能上传的图片与文件的大小，单位：MB。

更多站点附件设置说明与操作可查看：附件设置 (opens new window)。

# 内容过滤设置

内容过滤设置是对用户发布内容中非法、广告词、恶意灌水或骂人等的词语或者注册时用户名是否包含违法词语进行过滤，从而以保证站点内容的文明，合法。过滤词可以单独添加，也可以批量添加。详情操作可查看：内容过滤设置 (opens new window)

TIP

此处内容过滤设置主要指使用 Discuz! Q 系统自带的功能对内容进行检查，过滤程度与效果主要取决于设置的过滤词。同时设置内容过滤也是发布微信小程序时的审核项，良好的内容过滤有助您通过审核与核查。同时您也可以设置 Discuz! Q 内置的腾讯云图片内容安全功能，调用腾讯云 T-Sec 天御 图片内容安全 (opens new window)服务，精准识别令人反感、不安全或不适宜内容，节省用户运营成本。

# 第三方登录设置

用户通常在每个网站都会注册一个帐号，很多时候自己常用的用户名会存在已经被占用的情况，随着注册的帐号越来越多，导致用户的帐号管理越来越繁琐，且靠死记硬背已经不太现实。微信登录很好的解决了这个问题，且目前已经成为我们最常用的登录方式，用户只需绑定自己常用的体系帐号即可。

Discuz! Q 目前内置了以下三种微信登录三种不同的登录方式：公众号、小程序、PC端。如下图所示：

# 腾讯云配置

Discuz! Q 集成了多项腾讯云云服务，其中包括图片内容安全、文本内容安全、短信、实名认证、对象存储、视频等。您可以根据自己的实际需求以及简单的配置，即可与腾讯云云服务协同使用，帮助您节省大量的人力成本和快捷的调用云服务功能。
