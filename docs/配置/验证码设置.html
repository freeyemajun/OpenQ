    # 验证码设置

    # 前提条件

    • 已设置并开启云API (opens new window)。

    # 操作指南

    TIP

    • 设置证码请先配置云API和已 开通验证码服务 (opens new window) 服务，并确保有对应资源包。
    • 资源包购买您可前往 验证码购买页 (opens new window) 进行购买。
    1. 使用 http(s)://{站点名称}/admin 登录 Discuz! Q 站点后台。
    2. 在 Discuz! Q 站点后台中依次单击【全局】>【腾讯云设置】。
    3. 单击 “验证码” 处【配置】。如下图所示：

    1. 在验证码配置页，填写相关配置信息。如下图所示：

    • APPID：图形验证的 CaptchaAppId 参数。获取方式参见：验证码各参数填写说明 (opens new window)。
    • APP Secret Key：图形验证的 CaptchaAppId 参数。获取方式参见：验证码各参数填写说明 (opens new window)。
    1. 完成信息填写后，单击【提交】并返回腾讯云设置页面。
    2. 在腾讯云设置页面，验证码处单击【开启】。如下图所示：

    1. 设置页面弹出“修改成功”后，即可完成验证码设置操作。如下图所示：

    # 验证码各参数填写说明

    APPID与AppSecretKey您可前往腾讯云验证码控制台 (opens new window)进行获取。如下图所示：

    TIP

    • 若您有多个图形验证，您可选择其中1个进行填写。若无您可【新建验证】。

    # 新建验证操作如下：

    1. 在腾讯云验证码控制台 (opens new window)，依次单击【图形验证】>【新建验证】。
    2. 在弹出的“新建验证”窗口中，填写以下信息。如下图所示：

    验证名称：验证码名称，可自定义填写。 验证所属域名：需要调用该验证码的域名，可自定义填写。（域名仅用于标识场景，无调用或验证作用。） 验证渠道：选择【Web端/App(IOS或Android)】。 验证场景：选择【账号场景(登录、注册等)】即可新建成功。
