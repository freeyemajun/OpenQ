    # 短信设置

    # 概述

    用户的密码取回、提现时都依托于短信验证，Discuz! Q 内置了腾讯云“短信”服务。本文将指导您如何设置并开启 Discuz! Q “短信” 设置。

    # 前提条件

    • 已设置并开启云API (opens new window)。

    # 操作指南

    TIP

    • 设置短信请先配置云API，并确保腾讯云账户的短信额度充足。
    • 短信额度购买您可前往 Discuz! Q 活动页 (opens new window) 进行购买。
    1. 使用 http(s)://{站点名称}/admin 登录 Discuz! Q 站点后台。
    2. 在 Discuz! Q 站点后台中依次单击【全局】>【腾讯云设置】。
    3. 单击 “短信” 处【配置】。如下图所示：

    1. 在 “短信” 配置页面，请填写相关配置信息。如下图所示：

    # SDK AppID 信息获取

    1. 登录短信控制台 (opens new window)。
    2. 在【应用管理】>【应用列表】处，即可获取 SDK AppID 信息。如下图所示：

    TIP

    • 若您有多个 SDK AppID ，选择其一即可。
    • 若无 SDK AppID ，单击【创建应用】填写相关信息后即可获取。

    # App Key 信息获取

    1. 登录短信控制台 (opens new window)。
    2. 在【应用管理】>【应用列表】处，单击设置 Discuz! Q 的 SDK AppID 卡片。
    3. 在应用信息处，即可获取 App Key 信息。如下图所示：

    # 短信验证码使用模板ID：

    1. 登录短信控制台 (opens new window)。
    2. 在【国内短信】>【正文模板管理】处，即可获取短信验证码使用模板ID。如下图所示：

    TIP

    • 若您的【正文模板管理】处无对应的模板，请先进行创建并审核通过。详情操作参见：管理国内短信正文模板 (opens new window)。
    • 模板内容需为两个参数，{1}与{2}。建议使用标准模板【{1}为您的登录验证码，请于{2}分钟内填写，如非本人操作，请忽略本短信。】

    # 短信签名：

    1. 登录 短信控制台 (opens new window)。
    2. 在【国内短信】>【签名管理】处，即可获取短信签名。如下图所示：

    TIP

    • 若您的【签名管理】处无对应签名，请先进行创建并审核通过。详情操作参见：管理国内短信签名 (opens new window)。
    • 签名为信息栏【内容】处信息，非 ID。
    1. 信息填写完成后，单击【提交】。
    2. 返回腾讯云设置页面，在【短信】处单击【开启】。如下图所示：

    1. 设置页面弹出“修改成功”后，即可完成短信设置操作。如下图所示：
