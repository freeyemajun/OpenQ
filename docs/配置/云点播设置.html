    # 云点播设置

    # 概述

    Discuz! Q 视频和语音依托于腾讯云的云点播服务，只有当此处配置并开启之后，用户在前台才可以进行视频和语音内容的发布。本文将指导您如何设置云点播。

    # 前提条件

    • 已设置并开启云API (opens new window)。

    # 操作指南

    TIP

    • 设置云点播请先配置云API和已开启 云点播VOD服务 (opens new window) 服务，并确保有对应资源包。
    • 资源包购买您可前往 Discuz! Q 活动页 (opens new window) 进行购买。详细操作参见：购买资源包 (opens new window)。
    1. 使用 http(s)://{站点名称}/admin 登录 Discuz! Q 站点后台。
    2. 在 Discuz! Q 站点后台中依次单击【全局】>【腾讯云设置】。
    3. 单击 “云点播” 处【配置】。如下图所示：

    1. 在云点播配置页，填写相关配置信息。如下图所示：

    1. 完成信息填写后，单击【提交】并返回腾讯云设置页面。

    TIP

    信息填写详情说明参见：云点播各参数填写说明 (opens new window)

    # 云点播各参数填写说明

    # 子应用 ID：

    • 如果没有开启子应用，此处留空即可，留空将自动采用主应用。开通子应用详情参见：开通子应用 (opens new window)。
    • 如果您已开启子应用，您可参见 查看子应用 ID 信息 (opens new window)获取ID 信息。

    # 转码模板：

    • 云点播将会预设多个转码模板，您可选择其一进行填写。详情参见：查看转码模板 ID 信息 (opens new window)。
    • 如果没有您需的转码模板，请先创建对应模板。详情参见：模板设置 (opens new window)。

    # 水印模板：

    您可根据业务需要通过上传图片的方式来创建水印模板，并进行自定义的水印配置以及水印在视频中的位置设置。

    • 云点播将会预设1个默认水印模板，您可选择使用该默认水印模板进行填写。详情参见：查看水印模板 ID 信息 (opens new window)。
    • 如果没有您需的水印模板，请先创建对应水印模板。详情参见：模板设置 (opens new window)。

    # 截图模板：

    您可根据业务需要创建新的截图模板，并对上传的视频进行多种方式的截图，目前控制台支持时间点截图、采样截图和雪碧图截图三种截图方式。

    • 云点播将会预设多个默认截图模板，您可选择其一进行填写。详情参见：查看截图模板信息 (opens new window)。
    • 如果没有您需的截图模板，请先创建对应截图模板。详情参见：模板设置 (opens new window)。

    # 动图封面任务流名称：

    任务流设置可以通过创建的模板，流程化地对视频进行转码、添加水印、截图及视频智能识别等操作。

    • 云点播将会预设多个默认任务流名称，您可选择其一进行填写。详情参见：查看任务流名称信息 (opens new window)。
    • 如果没有您需的任务流名称，请先创建对应任务流名称。详情参见：任务流设置 (opens new window)。

    # 云点播防盗链 Key:

    云点播防盗链主要用于视频播放的权限控制，可有效防止视频资源盗刷。

    • 云点播将会预设1个默认分发域名，您可选择使用该默认分发域名进行填写。详情参见：查看云点播防盗链 Key 信息 (opens new window)。
    • 如果默认分发域名不满足您的要求，您可先添加分发域名后，在设置防盗链 Key。详情参见：自定义域名 (opens new window)。
    • 设置防盗链 Key 详情参见：Key 防盗链 (opens new window)。

    # 云点播防盗链签名有效期：

    单位秒，用于校验防盗链 URL 的合法性，过期后该 URL 将不再有效，返回 403 响应码。考虑到机器之间可能存在时间差，防盗链 URL 的实际过期时间一般比指定的过期时间长 5 分钟，即额外给出 300 秒的容差时间。建议过期时间戳不要过短，确保视频有足够时间完整播放。 例如： 若您的 Discuz! Q 站点的视频最大时长不超过 10 分钟，建议您设置数值为 600。依次类推。

    # 支持的视频扩展名:

    上传视频文件的扩展名，建议设置为 wmv,rm,mov,mpeg,mp4,3gp,flv,avi,rmvb。

    # 支持的最大尺寸:

    指上传视频文件的文件大小，单位：MB。您可根据您的 Discuz! Q 站点自定义。

    # 云点播回调校验码:

    ::tip 强烈建议您配置回调校验码，否则将可能导致他人可以随意的删除您站点上的任意视频贴和附件。 :::

    指云点播回调时的校验码，用于回调时的安全验证。您可根据您的具体情况进行自定义。校验码建议填写8位以上中英文字符。 填写完云点播校验码后，还需配置到云点播回调设置中。详情操作如下：

    # 回调设置

    1. 根据您的 Discuz! Q 站点域名与校验码拼接回调URL。拼接格式如下：
    http(s)://当前域名/api/threads/notify/video?qvodtoken=云点播回调校验码

    例如：您的 Discuz! Q 站点域名为 https://discuz.chat/,云点播回调校验码为 password1234567,则回调URL为https://discuz.chat/api/threads/notify/video?qvodtoken=password1234567。

    1. 登录云点播控制台 (opens new window)，依次单击【回调设置】>【设置】，并在【3.0 版本格式回调 URL】填写您的回调URL与勾选事件通知【频上传完成】、【视频删除完成】。如下图所示：

    1. 单击【确定】，即可完成回调设置。
