# 微信小程序升级

概述

本文档将指导您如何升级 Discuz! Q 微信小程序。

TIP

Discuz! Q 会定期更新 Discuz! Q 微信小程序版本，您可以前往 更新记录 (opens new window) 查看更新内容，并根据需求决定是否更新。

前提条件

  • 更新记录 (opens new window) 已更新微信小程序内容。

操作指南

  1. 更新记录 (opens new window) 已更新微信小程序内容后，您可以单击此处 (opens new window)重新下载小程序源码，并进行微信小程序 构建 (opens new window)。

TIP

微信小程序版本更新需等待微信审核通过，请耐心等待。

  1. 微信小程序构建成功后，可参考 发布微信小程序 (opens new window)，进行版本发布。
